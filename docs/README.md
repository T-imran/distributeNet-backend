## Quick Summary

DistribuNet is a multi-tenant distribution management dashboard built with Next.js, React, TypeScript, Tailwind CSS, React Query, Zustand, and mock in-memory data services.

The product is structured around:

- authentication and tenant context
- executive dashboard analytics
- dealer, retailer, salesman, order, payment, and product operations
- reporting and AI insight views
- tenant configuration, user management, and notifications
